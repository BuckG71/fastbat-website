# FASTBAT — Trinity University Baseball

Official website for FASTBAT (Friends and Supporters of Trinity Baseball), a 501(c)3 nonprofit booster organization supporting Trinity University Baseball in San Antonio, TX.

## Development

Serve locally with any static file server:

```bash
python3 -m http.server 8080
```

Then open http://localhost:8080

## Deployment

This site deploys automatically to GitHub Pages on every push to `main` via GitHub Actions.

To enable GitHub Pages:
1. Go to **Settings > Pages** in your GitHub repo
2. Under **Build and deployment**, select **GitHub Actions** as the source
